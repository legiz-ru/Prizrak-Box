Диплинки позволяют открыть Prizrak-Box и автоматически заполнить URL подписки одним нажатием.
Это удобно для передачи ссылок пользователям без ручного ввода.

![Пример диплинка](../../../../assets/deep-linking-example.webp)

## Формат URL

```
prizrak-box://install-config?url=https://example.com/sub/username
```

## Как использовать

<Steps>
1. Поделитесь ссылкой `prizrak-box://install-config?url=...` с пользователями (в Telegram, на сайте, в QR-коде и т.д.).
2. При нажатии на ссылку на устройстве с установленным Prizrak-Box:
   - **Android** — система предложит открыть ссылку в Prizrak-Box.
   - **Windows/macOS/Linux** — обработчик протокола откроет приложение автоматически.
3. Откроется диалог **Добавить подписку** с предзаполненным URL.
4. Пользователь подтверждает — подписка добавлена.
</Steps>
