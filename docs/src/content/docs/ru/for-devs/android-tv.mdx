## pxtv — импорт со страницы подписки

![pxtv в действии](../../../../assets/android-tv-pxtv.webp)

**[pxtv](https://github.com/legiz-ru/pxtv)** — вспомогательное веб-приложение, созданное специально для Prizrak-Box на Android TV.

Главная задача: **дать пользователям возможность импортировать подписку прямо со страницы провайдера** — без необходимости вводить длинные URL на клавиатуре телевизора.

Принцип работы:
1. Откройте pxtv на телефоне (работает как обычная веб-страница — установка не нужна).
2. Введите или нажмите ссылку на подписку на телефоне.
3. pxtv отправляет ссылку на телевизор через локальную Wi-Fi сеть.
4. Prizrak-Box на телевизоре получает и добавляет подписку автоматически.
