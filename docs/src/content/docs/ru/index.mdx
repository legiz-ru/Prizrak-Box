## Что такое Prizrak-Box?

Prizrak-Box — это графический прокси-клиент на базе [Mihomo](https://github.com/MetaCubeX/mihomo) (Clash.Meta).
Поддерживает Clash YAML и прямые ключи (vless://, hy2://), встроенные шаблоны правил для России и Китая, мониторинг трафика и TUN-режим.
