## Частые вопросы

<details>
<summary>**Что такое прокси-группа Smart?**</summary>

Данная группа использует адаптивный выбор узлов на основе весов. Подробнее ознакомиться с функционалом группы и внедрением в конфигурацию можно в [данном разделе](/ru/for-devs/smart-proxy-group/).

</details>

<details>
<summary>**Как GUI-настройки контроля доступа приложений взаимодействуют с `tun.include/exclude-package` из конфигурации? (Android)**</summary>

Приложение объединяет списки из конфига и GUI, но Android не поддерживает одновременно режимы include и exclude — если итоговый include непустой, exclude целиком игнорируется.

| Конфиг | GUI | Результат |
|--------|-----|-----------|
| `exclude-package` | исключить выбранные | ✅ GUI дополняет конфиг |
| `include-package` | разрешить только выбранные | ✅ GUI дополняет конфиг |
| `exclude-package` | разрешить только выбранные | ⚠️ В туннель идут только выбранные приложения в GUI |
| `include-package` | исключить выбранные | ⚠️ В туннель идут только выбранные приложения в конфигурации профиля |
| любой | принимать все | Управляет только конфиг |

**Правило:** одинаковые направления — дополняют друг друга; разные направления — include всегда перекрывает exclude. Это ограничение Android VpnService.

</details>

<details>
<summary>**Чем отличается режим прокси и режим TUN?**</summary>

**Режим прокси** работает на уровне приложений — трафик перехватывается только у тех программ, которые поддерживают системный прокси (HTTP/SOCKS). По умолчанию Prizrak-Box автоматически выставляет системный прокси на весь компьютер. Если это не нужно и вы хотите вручную настроить прокси в требуемых приложениях — отключите его в **Настройках приложения**.

**TUN-режим** создаёт виртуальный сетевой интерфейс и перехватывает весь трафик устройства на уровне пакетов — независимо от того, поддерживает ли приложение прокси. Требует прав администратора. При установке приложения вместе с ним устанавливается фоновый сервис — если он запущен, в дальнейшем запускать приложение с правами администратора не нужно: необходимые права предоставляются через сервис.

**Когда что выбирать:**
- Прокси — для повседневного использования в браузере и стандартных приложениях
- TUN — когда нужно пустить через прокси всё, включая игры, системные службы, системы с голосовыми чатами (например Discord) и приложения без поддержки прокси

</details>

<details>
<summary>**Как вместо кружочков доступности узлов узнать значение тестов задержки? (Android)**</summary>

1. В настройках приложения → раздел **Общие** → **Приложение** → переключите **Отображение задержки (Цветные точки)** в положение «выключено».

2. Если вы владелец сервера — вы можете централизованно распространить числовые значения задержек для всех пользователей через HTTP-заголовок [`pxa-latency-dots`](/ru/for-devs/supported-headers/#управление-интерфейсом-android).

</details>
