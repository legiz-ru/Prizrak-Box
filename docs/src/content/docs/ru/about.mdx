## Обзор

**Prizrak-Box** — это кроссплатформенный прокси-клиент с открытым исходным кодом, работающий на базе [Mihomo](https://github.com/MetaCubeX/mihomo) (бывший Clash.Meta).
Приложение предоставляет удобный графический интерфейс для управления прокси-подписками, мониторинга трафика и настройки правил маршрутизации.

## Управление подписками

![Управление профилями](../../../assets/about-profiles.webp)

Prizrak-Box поддерживает три типа профилей:

1. **Удалённая подписка** — загружается по URL с возможностью автообновления
2. **Локальная конфигурация** — локальный Clash YAML-файл
3. **Агрегация** — объединение из нескольких источников
4. **Ключи подключений** — прямые ключи (`vless://`, `hy2://` и другие) со встроенными настраиваемыми шаблонами

При загрузке удалённой подписки сервер может возвращать HTTP-заголовки для управления поведением клиента — см. [Поддерживаемые заголовки](/ru/for-devs/supported-headers/).

## Архитектура

Приложение состоит из:

- **Wails v3** (Windows, macOS) — нативная оболочка, отображающая Vue 3 интерфейс
- **Electron** (Linux) — оболочка, отображающая Vue 3 интерфейс
- **Android** — нативная activity, отображающая Vue 3 интерфейс
- **Go-бэкенд** (`px`) — HTTP API, управление подписками, хранилище BoltDB
- **TUN-сервис** (`px-service`) — привилегированный системный сервис для TUN-режима (Windows, Linux)
- **Ядро Mihomo** — движок прокси-маршрутизации (кастомный форк), встроенный в Go-бинарник
