Диплинки позволяют открыть Prizrak Box и автоматически заполнить URL подписки одним нажатием.
Это удобно для передачи ссылок пользователям без ручного ввода.

![Пример диплинка](../../../assets/deep-linking-example.webp)

## Формат URL

```
prizrak-box://install-config?url=<закодированный-url-подписки>
```

Параметр `url` должен быть **URL-закодирован**.

### Пример

Для добавления подписки `https://example.com/sub/username`:

```
prizrak-box://install-config?url=https%3A%2F%2Fexample.com%2Fsub%2Fusername
```

## Как использовать

<Steps>
1. Сформируйте диплинк, URL-закодировав адрес подписки.
2. Поделитесь ссылкой `prizrak-box://install-config?url=...` с пользователями (в Telegram, на сайте, через QR-код и т.д.).
3. При нажатии на ссылку на устройстве с установленным Prizrak Box:
   - **Android** — система предложит открыть ссылку в Prizrak Box.
   - **Windows/macOS/Linux** — обработчик протокола откроет приложение автоматически.
4. Откроется диалог **Добавить подписку** с предзаполненным URL.
5. Пользователь подтверждает — подписка добавлена.
</Steps>

## Генерация диплинка

Используйте любой URL-энкодер. На JavaScript:

```js
const subscriptionUrl = 'https://example.com/sub/username';
const deepLink = `prizrak-box://install-config?url=${encodeURIComponent(subscriptionUrl)}`;
console.log(deepLink);
// prizrak-box://install-config?url=https%3A%2F%2Fexample.com%2Fsub%2Fusername
```

## Использование в HTML

```html
<a href="prizrak-box://install-config?url=https%3A%2F%2Fexample.com%2Fsub%2Fusername">
  Добавить подписку в Prizrak Box
</a>
```

## QR-код

Диплинк `prizrak-box://` можно закодировать в QR-код — пользователь сканирует его на Android, и система предлагает открыть в Prizrak Box.
