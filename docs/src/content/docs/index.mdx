## What is Prizrak Box?

Prizrak Box is a graphical proxy client that uses the [Mihomo](https://github.com/MetaCubeX/mihomo) (Clash.Meta) core.
It supports remote subscriptions, automatic updates, traffic monitoring, and advanced routing rules.
