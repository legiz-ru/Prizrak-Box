## What is Prizrak-Box?

Prizrak-Box is a graphical proxy client that uses the [Mihomo](https://github.com/MetaCubeX/mihomo) (Clash.Meta) core.
Supports Clash YAML and direct proxy keys (vless://, hy2://), built-in routing templates for Russia and China, traffic monitoring, and TUN mode.
