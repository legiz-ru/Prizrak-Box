<Aside type="caution" title="Read before use">
Prizrak Box is a **proxy client** — it connects to proxy servers configured in your subscription.
The app itself does not provide any proxy servers or bypass services.
</Aside>

## Intended Use

Prizrak Box is intended for:

- **Privacy** — routing your traffic through a trusted proxy to protect your data on public networks.
- **Development & Testing** — inspecting and routing traffic during development.
- **Access** — reaching resources that are restricted in your network.

## Legal Disclaimer

## Security Notes

<Aside type="tip" title="Subscription security">
Only add subscriptions from sources you trust. A malicious subscription server could return harmful routing rules.
</Aside>

- Subscription URLs may include credentials — keep them private and do not share them publicly.
- When using **HWID authentication** (`X-Hwid-Active`), your hardware ID is sent to the subscription server. Be aware of this if privacy is a concern.
- **Age-encrypted** subscriptions protect your configuration content in transit; the secret key is stored locally in the app's profile storage.

## Data Storage

All profile data (subscription URLs, credentials, settings) is stored locally in the application's data directory using [BoltDB](https://github.com/etcd-io/bbolt). No data is sent to any third-party service by the application itself.

## Updates

</Aside>
