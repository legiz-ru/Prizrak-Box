## Data Collection

Prizrak-Box **does not collect or store** any data about your traffic or proxy connections.

The app connects to proxy servers that users configure themselves via subscriptions. Responsibility for data security on the proxy server side rests with the server owner.

## Data Sent to the Subscription Server

When fetching a subscription, Prizrak-Box may automatically send the following HTTP headers to the server:

- `X-Hwid` — hardware device identifier (if the server uses HWID Device Limit)
- `X-Device-OS` — operating system name
- `X-Ver-OS` — OS version
- `X-Device-Model` — device model

Transmission of the HWID identifier can be disabled in the app settings.

## Local Data Storage

All profile data (subscription URLs, keys, settings) is stored locally on the device using [BoltDB](https://github.com/etcd-io/bbolt). No data is sent to any third-party service by the application itself.

## Monetization and Advertising

Prizrak-Box does not use monetization or advertising and has no plans to introduce them.

## Children

The app does not collect any personally identifiable information from anyone, including children under the age of 13.

## Third-Party Links

The documentation and the app may contain links to third-party resources. Prizrak-Box is not responsible for the content or privacy practices of third-party sites.

## Policy Updates

Changes take effect upon publication. We recommend reviewing this page periodically.
